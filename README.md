# 🛡️ My Personal Security Tools

Welcome to my Cybersecurity and IT Automation portfolio! This repository contains a collection of Python scripts that I built while learning about Network Security, Linux, and Automation.

## 🛠️ Tools Included

### 1. Secure Password Generator (`pass_gen.py`)
A script that generates highly secure, randomized passwords of any length to protect against brute-force attacks.
* **Features:** Uses uppercase, lowercase, numbers, and special characters.

### 2. Simple Network Scanner (`network_scanner.py`)
A Ping Sweep automation tool that scans the local network to discover active devices.
* **Features:** Scans up to 255 IPs automatically and ignores inactive devices (Firewall evasive).
